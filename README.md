## To Play

Just run `python3 pycalc.py`

## Features : 

#### Version1.0

* Cross-platform
* Easy to use
* Supports common mathematical functions like 

| `+`   | `-`  | `*`  | `/`  |  `%` |

* Supports `Random` and `AC`
